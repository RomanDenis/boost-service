# Проект Frontend для Сервиса Бустинга

Этот проект представляет собой frontend-архитектуру для сервиса игрового бустинга, разработанную на Next.js с использованием App Router. Архитектура спроектирована с учетом масштабируемости, SEO-оптимизации и удобства поддержки.

## Технологический стек

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Architecture**: Feature-based, modular monolith

## Структура проекта

Проект разделен на несколько ключевых каталогов, каждый из которых выполняет свою роль.

### `app`

Здесь находится вся логика маршрутизации и определения страниц, согласно принципам App Router в Next.js.

- **`(public)`**: Публичные страницы (Home, FAQ, Pricing и т.д.).
- **`(auth)`**: Страницы аутентификации (Login, Register).
- **`(user)`**: Личный кабинет пользователя (Dashboard, Orders).
- **`(admin)`**: Панель администратора.
- **`games/[game]`**: Динамические маршруты для каждой игры и её услуг (boost, coaching).
- **`layout.tsx`**: Корневой layout приложения.
- **`globals.css`**: Глобальные стили.

### `features`

Содержит основную бизнес-логику приложения, сгруппированную по функциональным модулям.

- **`auth`**: Логика аутентификации и управления сессией.
- **`boost`**: Ядро калькулятора цен, включая конфигурации рангов, цен и модификаторов для каждой игры.
- **`order`**: Управление заказами.
- **`user`**: Логика, связанная с данными пользователя.
- **`payment`**: Интеграция с платежными системами.

### `components`

Переиспользуемые UI-компоненты, которые не содержат бизнес-логики.

- **`ui`**: Атомарные компоненты (Button, Input, Card).
- **`shared`**: Более сложные, составные компоненты (Header, Footer, PriceSummary).

### `lib`

Вспомогательные утилиты, хуки и константы, используемые в разных частях приложения.

- **`utils.ts`**: Общие функции.
- **`hooks`**: Кастомные хуки (например, `use-local-storage`).
- **`constants.ts`**: Глобальные константы.

### `types`

Определения TypeScript-типов для всего приложения.

- **`index.ts`**: Общие типы.
- **`games.ts`**: Типы, специфичные для игр (ранги, услуги).

### `styles`

Глобальные стили и конфигурация темы Tailwind CSS.

### `public`

Статические ассеты: изображения, иконки, шрифты.
